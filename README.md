# ionic-vue

### インストール
```
npm install
```

### 起動
```
npm run serve
```

### ビルド
```
npm run build
```

## Capacitor

### Add Platforms
```
npx cap add ios
npx cap add android
```

### Open IDE to build
```
npx cap open ios
npx cap open android
```

### Syncing your app with Capacitor
```
npx cap copy
```
